隐私政策

南京云玑信息科技有限公司

南京云玑信息科技有限公司严格遵守法律法规，遵循以下隐私保护原则，为您提供更加安全、可靠的服务：请花一些时间熟悉我们的隐私惯例，如果您有任何问题，请告诉我们。

本隐私政策规定了南京云玑信息科技有限公司及其关联公司（下文简称“我们”）如何收集、使用、披露、处理和保护您在使用产品和服务时提供的信息。

通过使用我们的产品和服务，即表示您已阅读、认可并接受本隐私政策及其更新。若您对本政策有任何疑问，请通过客服电话 025-52388619 联系我们。

收集哪些信息以及如何使用？

收集的信息类别

为了向您提供我们的服务，我们会要求您提供必要的个人信息。若不提供，我们可能无法提供相关功能。

  • 您提供或上传的信息： 包括联系方式（手机号、账号、头像等）、联系人号码、消息、反馈等。
  • 我们指定的与您相关的信息： 例如软件账号。
  • 第三方服务提供商提供的信息： 如设备 ID。
  • 设备信息： 如 MAC 地址、系统版本、硬件信息等。
  • 应用信息： 如 SDK 版本、系统更新设置。
  • 位置信息： 城市代码、经纬度等（仅适用于特定功能）。
  • 登录日志信息： 包括 cookie、IP 地址、网络请求、系统日志、崩溃信息。

信息的使用目的

收集信息的主要目的是为您提供和改进产品/服务，并遵守法律。我们可能会将您的信息用于以下目的：

  • 提供、维护、改进产品和服务（包括售后和客户支持）。
  • 与您沟通（更新、咨询、支持）。
  • 分析和开发统计信息，以优化产品。
  • 存储并维护相关信息，用于业务运营或法律合规。
  • 提供本地服务而无需与服务器通信。
  • 创建用户账号和资料页。
  • 提供基于位置的服务（如地图、天气）。
  • 推送通知或推广信息。
  • 验证用户身份，防止未经授权访问。
  • 收集用户反馈，用于改进服务。
  • 发送重要通知（如条款或政策变更）。
  • 设备分析以提升用户体验。

信息共享

我们不会出售任何个人信息，但可能在以下情况下与第三方共享：

  • 为提供您所需的产品/服务，与第三方服务供应商和关联公司共享。
  • 在法律要求下，向相关机构披露。

第三方 SDK 使用说明

  • 地图 SDK： 高德公司，收集设备和位置信息，用于地图导航。隐私政策

  • 直播 SDK（Android）： 腾讯公司，收集设备/网络信息。隐私政策
  • 短视频 UGSV SDK： 腾讯公司，收集设备信息、视频文件。隐私政策
  • 设备标识统一调用 SDK： 中国信通院，收集 OAID/VAID 等。隐私政策
  • 腾讯云即时通信 IM SDK： 腾讯云，收集账号、设备、消息内容。隐私政策
  • 轻量版地图 SDK： 高德公司，收集设备/位置信息。隐私政策
  • TRTC 音视频通话 SDK： 腾讯公司，收集设备/网络/音视频数据。隐私政策

安全保障

我们采取合理的物理、电子和管理措施保护您的信息，存储在安全的服务器和受控设施中，并签订保密协议以确保第三方合规。但需注意，互联网传输存在风险，我们无法保证绝对安全。

您能做什么

  • 妥善保管账号和密码，不向未经授权人员泄露。
  • 若发现账号被盗或存在安全漏洞，请立即通知我们。
  • 积极配合我们的安全措施。

保留政策

我们会在达成收集目的或遵守法律要求的必要期限内保留您的个人信息。一旦无保留必要，我们将删除或匿名化处理。

用户权利

访问与更正

您有权访问、更正您的个人信息。在处理请求前，我们会验证身份，并在法定时间内回复。
